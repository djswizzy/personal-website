# David Calvert Personal Website

This is my personal website built with Jekyll.

This website was created as a fork of [academicpages/academicpages.github.io](https://github.com/academicpages/academicpages.github.io)

## Setup

1. Install Ruby and Jekyll
2. Run `bundle install`
3. Run `bundle exec jekyll serve` to start the local development server
4. Visit `http://localhost:4000` to view the site

## Customization

- Edit `_config.yml` for site-wide settings
- Edit pages in `_pages/` for content
- Add blog posts in `_posts/`
- Update `_data/cv.json` for CV information
